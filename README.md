## üåü Project Overview

**MindPulse AI** is an intelligent mental health monitoring and stress prediction system designed for students.  
It analyzes both **academic & lifestyle behavior** and **emotional journal inputs** to predict stress levels and provide personalized wellness recommendations.

This system acts as an early-warning mental wellness assistant for educational institutions.
